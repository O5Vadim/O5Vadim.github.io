# 🚀 Мой персональный сайт на GitHub Pages

Добро пожаловать в мой репозиторий! Это мой личный проект, где я экспериментирую с современными веб-технологиями.

## ✨ Основные фишки проекта:

*   **🛠 Свой Service Worker:** Сайт работает даже **без интернета** (Offline-mode). Все важные страницы кэшируются при первом посещении.
*   **🎨 Кастомная страница 404:** Красивый дизайн с градиентом и удобной кнопкой возврата, если файл не найден.
*   **📱 PWA Ready:** Сайт оптимизирован для мобильных устройств.
*   **⚡ Быстрый деплой:** Автоматическая сборка через GitHub Actions.
